Emargator – Dépôt APT

Ce dépôt APT fournit deux paquets :

  • emargator – version stable
  • emargator-beta – version bêta (pré-release)

Les deux paquets sont en conflit : un seul peut être installé à la fois.

Installation

1. Importer la clé GPG

curl -fsSL https://crazycat256.github.io/Emargator/public.key | sudo gpg --dearmor -o /usr/share/keyrings/emargator.gpg

2. Ajouter le dépôt

echo "deb [signed-by=/usr/share/keyrings/emargator.gpg] https://crazycat256.github.io/Emargator ./" \
  | sudo tee /etc/apt/sources.list.d/emargator.list

1. Ajouter le dépôt

echo "deb [trusted=yes] https://crazycat256.github.io/Emargator ./" \
  | sudo tee /etc/apt/sources.list.d/emargator.list

2. Installer

sudo apt update

# Version stable
sudo apt install emargator

# OU version bêta
sudo apt install emargator-beta

3. Mettre à jour

sudo apt update && sudo apt upgrade
Note : emargator et emargator-beta ne peuvent pas être installés en même temps. L'installation de l'un supprimera automatiquement l'autre.
